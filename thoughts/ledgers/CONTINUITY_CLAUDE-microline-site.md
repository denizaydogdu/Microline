# CONTINUITY — microline-site

## Goal
Türkçe-öncelikli (EN seçmeli) Microline vitrin + sipariş formu sitesi. Spring Boot 3.5 + Thymeleaf + yerel PostgreSQL 14 (Docker YOK), Maven Wrapper + JDK 21. "Bitti" = rota tablosundaki tüm sayfalar iki dilde çalışıyor, formlar DB'ye yazıyor, sitemap/hreflang doğru, testler yeşil.

## Constraints
- Docker kullanılmayacak — yerel Homebrew postgresql@14 (dev: `microline`, test: `microline_test`)
- JDK 21 zorunlu (sistem 17) — `export JAVA_HOME=$(/usr/libexec/java_home -v 21)`
- Sepet/ödeme/üyelik YOK (v2); şema engellemeyecek
- Türkçe varsayılan/fallback; `_tr` kolonlar NOT NULL
- Flyway şemanın sahibi (`ddl-auto=validate`); seed = prod içerik
- Tasarım: framework'süz CSS, 8px grid token'ları (CLAUDE.md'deki kurallar)

## Key Decisions
- URL-path tabanlı dil (`/tr/...`, `/en/...`), `PathLocaleResolver` + `PageRoute` enum + çift `@GetMapping` (SEO/hreflang için; cookie/?lang= reddedildi)
- `_tr`/`_en` kolon çiftleri (translation table reddedildi — 2 sabit dil)
- Yasal sayfalar DB'de, `/{lang:tr|en}/{slug}` fallback route
- Testcontainers yerine yerel `microline_test` DB (Docker yok)
- Spring Boot parent 3.5.7; Lombok yok, DTO = record
